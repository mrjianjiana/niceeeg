## 📄 核心文件详细说明

### 1. **主训练脚本**

#### `nice_stand.py` ⭐ **核心文件**
**作用**: 项目的主入口，包含完整的训练和测试流程

**主要功能**:
- **模型定义**:
  - `PatchEmbedding`: EEG信号的特征提取（时空卷积）
  - `Enc_eeg`: EEG编码器
  - `Proj_eeg`: EEG特征投影层
  - `Proj_img`: 图像特征投影层
- **训练流程**:
  - 加载EEG训练数据和图像特征
  - 使用对比学习（Contrastive Learning）训练模型
  - 将EEG特征和图像特征映射到同一嵌入空间
  - 使用余弦相似度计算损失
- **验证流程**:
  - 每个epoch后在验证集上评估
  - 保存最佳模型权重
- **测试流程**:
  - 加载最佳模型
  - 在测试集上计算Top-1, Top-3, Top-5准确率
  - 通过计算EEG特征与200个类别中心特征的相似度进行分类

**关键参数**:
- `--dnn`: 选择DNN模型（clip/vit/resnet）
- `--epoch`: 训练轮数
- `--num_sub`: 被试数量
- `--batch-size`: 批次大小

### 2. **数据预处理模块**

#### `preprocessing/preprocessing.py`
**作用**: EEG数据预处理的主脚本

**功能**:
- 调用预处理工具函数
- 对每个被试的EEG数据进行epoching、MVNN归一化
- 保存预处理后的数据

#### `preprocessing/preprocessing_utils.py`
**作用**: 预处理工具函数集合

**主要函数**:
- `epoching()`: 
  - 通道选择（63个通道）
  - 数据分段（epoching）
  - 基线校正
  - 重采样到250 Hz
  - 按图像条件排序
- `mvnn()`: 多变量噪声归一化（Multivariate Noise Normalization）
- `save_prepr()`: 保存预处理后的数据

**输出格式**: `(conditions, repetitions, channels, time_points)`

#### `preprocessing/get_center_images.py`
**作用**: 从原始Things数据集中获取每个测试条件的中心图像

## 🎯 项目目标

### 核心目标
**从EEG信号中解码自然图像，实现物体识别**

### 具体任务
1. **零样本分类**: 给定一段EEG信号，预测被试看到的图像属于200个物体类别中的哪一个
2. **跨模态对齐**: 将EEG信号和图像特征映射到同一语义空间
3. **自监督学习**: 使用对比学习，无需人工标注的类别标签

### 技术路线
```
EEG信号 → 特征提取 → 投影层 → 嵌入空间
                                    ↓
图像特征 → CLIP编码 → 投影层 → 嵌入空间
                                    ↓
                        对比学习（Contrastive Learning）
                                    ↓
                        最大化匹配对的相似度
```

### 创新点
1. **多维度分析**: 从时间、空间、频谱、语义四个维度解析大脑活动
2. **可插拔模块**: 提供自注意力（SA）和图注意力（GA）模块
3. **零样本性能**: 在大型数据集上实现显著的零样本分类性能

# 1. 加载最佳模型
self.Enc_eeg.load_state_dict(torch.load('./model/Enc_eeg_cls.pth'))

# 2. 对测试集EEG数据进行编码
tfea = self.Proj_eeg(self.Enc_eeg(teeg))
tfea = tfea / tfea.norm(dim=1, keepdim=True)  # 归一化

# 3. 计算与200个类别中心的相似度
similarity = (100.0 * tfea @ all_center.t()).softmax(dim=-1)

# 4. 获取Top-K预测
_, indices = similarity.topk(5)

# 5. 计算准确率

### 4. **验证指标说明**

#### Top-K准确率
- **Top-1**: 预测概率最高的类别是否等于真实类别
- **Top-3**: 真实类别是否在前3个预测中
- **Top-5**: 真实类别是否在前5个预测中

**意义**:
- Top-1: 精确匹配的准确率
- Top-3/Top-5: 模型是否能够识别出相关类别（即使不是最准确的）

- **Top-1准确率**: 约12-15%（随机猜测为0.5%）

## 🔑 关键概念

### 对比学习（Contrastive Learning）
- **目标**: 让匹配的EEG-图像对在嵌入空间中更接近，不匹配的更远
- **损失函数**: 交叉熵损失，基于余弦相似度

### 零样本分类（Zero-shot Classification）
- **含义**: 测试时，模型需要从200个类别中选择，但训练时没有使用这些类别的标签
- **实现**: 通过计算EEG特征与200个类别中心特征的相似度进行分类

### 中心特征（Center Features）
- **定义**: 每个类别（200个）的代表性特征
- **来源**: 从该类别所有图像的特征中提取（均值或单个代表图像）
- **用途**: 测试时作为"类别原型"与EEG特征进行匹配
